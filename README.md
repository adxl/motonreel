# Motoenreel

## How to launch the project

1.1 For Linux  
```$ ./install.sh```  

1.2 For MacOS (requires yarn)  
```$ npm install -g yarn```  
```$ ./install_macos.sh```  

2 - `$ docker-compose build --pull --no-cache`  
3 - `$ docker-compose up -d`  

4.1 - Client : `http://localhost:3000`  
4.2 - Server : `http://localhost:9000`
